Design Guide

Largeur et hauteur maximales

Utilisez ces classes pour définir la largeur et la hauteur maximales d'un élément.

Classes de largeur maximale

  • .max-w-1 - Largeur maximale de 1rem
  • .max-w-2 - Largeur maximale de 2rem
  • ...
  • .max-w-14 - Largeur maximale de 14rem

Les classes ci-dessus définissent la largeur maximale de l'élément en rem. Pour les appareils de taille moyenne et grande, vous pouvez utiliser des classes supplémentaires pour définir la largeur maximale à différentes tailles d'écran. Les classes suivantes seront appliquées aux écrans d'une largeur minimale de :

  • .max-w-sm-1 - Largeur maximale de 1rem
  • .max-w-sm-2 - Largeur maximale de 2rem
  • ...
  • .max-w-sm-14 - Largeur maximale de 14rem
  • .max-w-md-1 - Largeur maximale de 1rem
  • .max-w-md-2 - Largeur maximale de 2rem
  • ...
  • .max-w-md-14 - Largeur maximale de 14rem
  • .max-w-lg-1 - Largeur maximale de 1rem
  • .max-w-lg-2 - Largeur maximale de 2rem
  • ...
  • .max-w-lg-14 - Largeur maximale de 14rem

Classes de hauteur maximale

  • .max-h-1 - Hauteur maximale de 1rem
  • .max-h-2 - Hauteur maximale de 2rem
  • ...
  • .max-h-14 - Hauteur maximale de 14rem

Les classes ci-dessus définissent la hauteur maximale d'un élément avec les valeurs en rem allant de 1 à 14, pour les tailles "small" (576px et plus), "medium" (768px et plus) et "large" (992px et plus). Pour définir la hauteur maximale, utilisez la classe "max-h-" suivie de la valeur de hauteur en rem, par exemple :

<div class="max-h-4">
              ...
            </div>

De même, pour définir la hauteur maximale à différentes tailles d'écran, utilisez les classes de médias responsives comme suit :

<div class="max-h-6 max-h-md-8 max-h-lg-10">
              ...
            </div>

Cela définit la hauteur maximale à 6rem pour les écrans de taille inférieure à 576px, 8rem pour les écrans de taille comprise entre 576px et 768px, et 10rem pour les écrans de taille supérieure à 992px.

Largeur Maximale

Les classes ci-dessous définissent la largeur maximale d'un élément avec les valeurs en rem allant de 1 à 14, pour les tailles "small" (576px et plus), "medium" (768px et plus) et "large" (992px et plus). Utilisez la classe "max-w-" suivie de la valeur de largeur en rem pour définir la largeur maximale :

<div class="max-w-8">
              ...
            </div>

De même, pour définir la largeur maximale à différentes tailles d'écran, utilisez les classes de médias responsives comme suit :

<div class="max-w-10 max-w-md-12 max-w-lg-14">
              ...
            </div>

Cela définit la largeur maximale à 10rem pour les écrans de taille inférieure à 576px, 12rem pour les écrans de taille comprise entre 576px et 768px, et 14rem pour les écrans de taille supérieure à 992px.

Les classes .container

Les classes .container sont utilisées pour envelopper le contenu de la page et définir la largeur maximale des éléments à l'intérieur. Elles sont disponibles en plusieurs tailles pour s'adapter à différents écrans :

  • .container : largeur maximale de 95% pour les écrans de toutes les tailles.
  • .container-sm : largeur maximale de 95% pour les écrans supérieurs à 576px.
  • .container-md : largeur maximale de 95% pour les écrans supérieurs à 768px.
  • .container-lg : largeur maximale de 95% pour les écrans supérieurs à 992px.

Les classes de colonnes

Les classes de colonnes sont utilisées pour créer des colonnes à l'intérieur des conteneurs .row. Il y a 12 colonnes disponibles. Les classes de colonnes sont disponibles en plusieurs tailles pour s'adapter à différents écrans :

  • .col-auto : colonne avec une largeur automatique.
  • .col-{x} : colonne avec une largeur de x/12*100%. Exemple : .col-6 crée une colonne de 50% de la largeur du conteneur parent.
  • .col-sm-{x} : colonne avec une largeur de x/12*100% pour les écrans supérieurs à 576px.
  • .col-md-{x} : colonne avec une largeur de x/12*100% pour les écrans supérieurs à 768px.
  • .col-lg-{x} : colonne avec une largeur de x/12*100% pour les écrans supérieurs à 992px.

Utilisation de la classe .row

La classe .row est utilisée pour créer une rangée de colonnes. Elle utilise Flexbox pour aligner les colonnes horizontalement et pour permettre à la rangée de se plier (wrap) lorsque les colonnes atteignent la largeur maximale de leur parent. Voici comment l'utiliser :

Exemple :

<div class="row">
                <div class="col-6"></div>
                <div class="col-6"></div>
              </div>

Dans cet exemple, nous avons deux colonnes de taille égale qui sont alignées horizontalement. La classe .row enveloppe les deux colonnes et leur applique les propriétés Flexbox nécessaires pour les aligner correctement.

Il est important de noter que la classe .row doit toujours être utilisée avec des colonnes (classes .col-x) pour garantir un comportement cohérent.

Classes de colonnes

Le système de grille de ce framework est basé sur un nombre de colonnes configuré à l'aide de la variable $cols.

Pour utiliser les classes de colonnes, vous pouvez utiliser la classe .col-auto pour une colonne avec une largeur automatique, ou les classes .col-{nombre} pour spécifier une largeur de colonne fixe.

Les classes .col-sm-{nombre}, .col-md-{nombre}, et .col-lg-{nombre} vous permettent de définir des largeurs de colonnes spécifiques pour différents points de rupture en utilisant les media queries correspondantes.

Exemples d'utilisation

                  <div class="row">
                    <div class="col-6 col-sm-4 col-md-3 col-lg-2">
                      Contenu de la colonne
                    </div>
                    <div class="col-6 col-sm-4 col-md-3 col-lg-2">
                      Contenu de la colonne
                    </div>
                    <div class="col-12 col-md-6 col-lg-8">
                      Contenu de la colonne
                    </div>
                  </div>

Dans cet exemple, la première et la deuxième colonne auront une largeur de 6 colonnes sur les petits écrans et 4 colonnes sur les écrans plus grands, tandis que la troisième colonne occupera toutes les colonnes sur les petits écrans et la moitié de l'espace disponible sur les écrans moyens et grands.

Classes pour les marges

Les classes suivantes permettent de définir des marges sur les éléments :

Classes pour la marge globale (m-x)

Classe Propriétés CSS
.m-0 margin: 0 !important;
.m-1 margin: 0.25rem !important;
.m-2 margin: 0.5rem !important;
.m-3 margin: 1rem !important;
.m-4 margin: 1.5rem !important;
.m-5 margin: 3rem !important;

Les classes .mx-x et .my-x permettent de définir respectivement la marge horizontale et la marge verticale.

Classes pour la marge horizontale (mx-x)

Classe Propriétés CSS
.mx-0 margin-right: 0 !important; margin-left: 0 !important;
.mx-1 margin-right: 0.25rem !important; margin-left: 0.25rem !important;
.mx-2 margin-right: 0.5rem !important; margin-left: 0.5rem !important;
.mx-3 margin-right: 1rem !important; margin-left: 1rem !important;
.mx-4 margin-right: 1.5rem !important; margin-left: 1.5rem !important;
.mx-5 margin-right: 3rem !important; margin-left: 3rem !important;

Guide d'utilisation des classes

Classes de police

Les classes suivantes permettent de définir la famille de police :

  • .font-family-josefin : définir la police "Josefin"
  • .font-family-roboto : définir la police "Roboto"

Classes de mise en forme de texte

Les classes suivantes permettent de définir la mise en forme du texte :

  • .uppercase : mettre en majuscules
  • .bold : mettre en gras
  • .text-align-center : aligner le texte au centre
  • .text-align-justify : justifier le texte

Classes de positionnement

Les classes suivantes permettent de définir la position :

  • .position-relative : position relative

Classes de taille de police

Les classes suivantes permettent de définir la taille de la police :

  • .font-size-X : définir la taille de police en X où X est compris entre 6 et 28 (par exemple .font-size-12)
  • .font-size-sm-X : définir la taille de police en X pour les écrans d'au moins 576 pixels de largeur
  • .font-size-md-X : définir la taille de police en X pour les écrans d'au moins 768 pixels de largeur
  • .font-size-lg-X : définir la taille de police en X pour les écrans d'au moins 992 pixels de largeur

Classes de rayon de bordure

Les classes suivantes permettent de définir le rayon de la bordure :

  • .border-radius-X : définir le rayon de la bordure en X où X est compris entre 6 et 28 (par exemple .border-radius-12)

Classes d'overflow

Les classes suivantes permettent de définir le comportement en cas de débordement :

  • .overflow-hidden : cacher le débordement
  • .overflow-x-hidden : cacher le débordement horizontal
  • .overflow-y-hidden : cacher le débordement vertical

Classe de visibilité

Les classes de visibilité permettent de masquer ou afficher des éléments en fonction de la largeur de l'écran. Les classes ".lg-hidden" et ".lg-visible" sont disponibles pour masquer ou afficher des éléments pour les écrans de largeur supérieure à 992px.

Classe Description
.hidden Masque l'élément, mais conserve sa place dans le layout
.visible Affiche l'élément
.lg-hidden Masque l'élément pour les écrans de largeur supérieure à 992px
.lg-visible Affiche l'élément pour les écrans de largeur supérieure à 992px
